git log
参看提交的日志

常用连接的参数
--oneline		一行显示
$ git log --oneline
825da4d test -u command
b63a126 k8s with calico
0c21b0d add readme
-n 数字		显示最近几条提交记录
$ git log -n 1
commit 825da4d2169eb29d6aa133028c5c09ec88bbd219
Author: shili <shili_wymail@163.com>
Date:   Sun Jul 12 15:47:44 2020 +0800
...
--graph		以图像化显示提交记录
$ git log --all --graph
